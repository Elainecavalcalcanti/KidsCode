O que é Programação?

Programar é ensinar o computador a fazer coisas usando comandos simples.

O computador não pensa sozinho. Ele faz apenas o que a gente manda.

O computador precisa de instruções

O computador não pensa sozinho. Ele faz apenas o que a gente manda.

Passo a passo

Quando damos instruções em ordem, o computador consegue entender.

Exemplo: ligar o robô, mandar ele andar para frente e depois pedir para ele parar.

Na programação funciona assim

Cada passo é um comando. Se pular um passo, o resultado não funciona direito.

O que dá para criar?

Com programação podemos criar jogos, sites, aplicativos e muitas outras coisas legais.

Esse é mais um passo na sua jornada para aprender programação.
