COMANDOS BÁSICOS

Comandos são instruções que fazem o computador agir

Os comandos são instruções que dizem ao computador o que fazer.

Andar para frente

Esse comando faz o personagem se mover para frente.

Exemplo: dar 5 passos para frente.

Virar

O personagem pode virar para a direita ou para a esquerda.

Exemplo: virar à direita para mudar de caminho.

Repetir

Serve para fazer a mesma ação várias vezes.

Exemplo: pular 3 vezes seguidas.

Fazer escolhas

O computador pode escolher o que fazer em cada situação.

Exemplo: se chover, usar guarda-chuva.

Esperar

Esse comando faz o personagem esperar um pouco.

Exemplo: esperar 2 segundos antes de continuar.
